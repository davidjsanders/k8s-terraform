
This is the IP Address for the Load Balancer: ${lbip}

When accessing a load balancer based service, do two things:

1. For starting/stopping: export LBIP=${lbip}
2. To access <service>.${lbip}.xip.io

An example Ingress
==================
---
apiVersion: extensions/v1beta1
kind: Ingress
metadata:
  name: traefik-web-ui
  namespace: default
spec:
  rules:
  - host: nginx-frontend.${lbip}.xip.io
    http:
      paths:
      - path: /
        backend:
          serviceName: nginx-service
          servicePort: nginx-http

Enjoy
